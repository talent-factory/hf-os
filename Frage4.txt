Voraussetzung
Seitengröße (Page Size): 4 KB = 4096 Bytes
Virtuelle Adresse besteht aus:
Page Number (obere Bits)
Offset innerhalb der Seite (untere 12 Bits, weil 2¹² = 4096)


Vorgehen
Seitenzahl berechnen: page = virt_addr / 4096
Offset berechnen: offset = virt_addr % 4096
Frame aus dem Page Table hohlen (z. B. Seite 0 → Frame 2)
Phys. Adresse = frame * 4096 + offset

16384	4	0	1	4096
