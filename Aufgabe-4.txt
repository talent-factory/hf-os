# 12300₁₀ (0x300C)
Seite 3, Present = 1, PFN = 111
Phys. Adresse = 111 × 4096 + 12 = 28692


# 16384₁₀ (0x4000)
Seite 4, Present = 0 → Page Fault

# 32800₁₀ (0x8020)
Seite 8, Present = 1, PFN = 100
Phys. Adresse = 100 × 4096 + 32 = 16416

Begründung:
Virtuelle Adresse wird in Seite und Offset zerlegt. Die Seitennummer bestimmt den PFN. Ist die Seite nicht vorhanden (Present=0), folgt ein Page Fault.
